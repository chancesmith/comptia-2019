# What TCP/IP protocols does Nmap use?

1. Application layer : FTP, HTTP, SNMP, BOOTP, DHCP
1. Transport layer : TCP, UDP, ICMP, IGMP
1. Network layer : ARP, IP, RARP
1. Data link layer : SLIP, PPP
